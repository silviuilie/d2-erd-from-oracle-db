## D2 ERD from Oracle

### query
list :
 - tables (type)(triggers?)
 - views ? MVs ?
 - columns (type, nullable, constraint, fk/pk, index name and type)
 - FKs (table.column->table.column)

 - db URL (+creds)
 - table exclusions(comma separated) - default none
 - out d2 name 
 - img name
